# pycarsimlib

# Environmental setup
1. `cd /path/to/workingdir`
1. `git clone https://github.com/MizuhoAOKI/pycarsimlib.git`

# Install pycarsimlib from pip
You can also install pycarsimlib easily with pip.

`pip install pycarsimlib`

# Run an Example Script
```
$ cd examples
$ python runcarsim.py
```
